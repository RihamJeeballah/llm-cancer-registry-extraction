# LLM-based Cancer Registry Information Extraction

This repository contains the code for experiments, prompt design, evaluation, and statistical analysis for extracting structured cancer registry information using LLMs.

## Structure

- `experiments/`: field-specific extraction pipelines (grade, morphology, TNM, laterality)
- `prompts/`: prompt templates
- `evaluation/`: evaluation scripts (to be added)
- `statistical_analysis/`: bootstrap and hypothesis testing scripts (to be added)
- `docs/`: workflow diagrams

## Data Availability

Clinical data are not included due to privacy and ethical restrictions.
